✅ الحل السريع (5 دقائق):

   1️⃣  احصل على مفتاح مجاني من:
       https://openrouter.ai
       (سجّل → Keys → Create Key)

   2️⃣  افتح ملف .env

   3️⃣  عدّل هذا السطر:
       API_KEY=sk-or-v1-your-key-here
       (ضع مفتاحك بدلاً من your-key-here)

   4️⃣  أعد تشغيل التطبيق:
       python main.py
